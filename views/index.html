Image Search Abstraction Layer

A Free Code Camp API Project Created by Eric Tingleff

Instructions


To Search Images:

Add the following to the end of this page's URL:

/search/<search term>

To move to a different page within search results add this to the search URL:

?offset=<number of pages after page 1>

To View the Recent Searches History:

Add the following to the end of this page's URL:

/history


Example


A search for images of Nicolas Cage:

https://img-search-abstraction-layer-eric.glitch.me/search/nicolas cage

To see the second page of results:

https://img-search-abstraction-layer-eric.glitch.me/search/nicolas cage?offset=1

To see the recent search history:

https://img-search-abstraction-layer-eric.glitch.me/history
